# User Interface Module

## Mục đích

Xây dựng giao diện người dùng cho hệ thống hỏi đáp.

## Chức năng

- Nhập câu hỏi.
- Hiển thị câu trả lời.
- Hiển thị nguồn tham khảo.
- Hiển thị lịch sử hội thoại (nếu có).

## Dự kiến các tệp

```
ui/
│
├── app.py
├── components.py
└── README.md
```

## Công nghệ

- Streamlit
- HTML/CSS (nếu cần)
