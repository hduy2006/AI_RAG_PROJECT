# Evaluation Module

## Mục đích

Đánh giá chất lượng của hệ thống RAG.

## Chức năng

- Lưu bộ câu hỏi kiểm thử.
- So sánh kết quả với đáp án tham chiếu.
- Tính các chỉ số đánh giá.

## Dự kiến các tệp

```
evaluation/
│
├── questions.json
├── metrics.py
└── README.md
```

## Chỉ số đánh giá

- Accuracy
- Precision
- Recall
- F1-score
- Response Time
