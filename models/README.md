# Models Directory

## Mục đích

Lưu các mô hình AI được sử dụng trong dự án nếu chạy cục bộ.

## Nội dung

Có thể bao gồm:

- Mô hình Embedding.
- Mô hình LLM.
- Các tệp cấu hình.

## Lưu ý

Không nên đưa các mô hình có kích thước lớn lên GitHub. Thay vào đó, chỉ lưu hướng dẫn tải mô hình hoặc cấu hình cần thiết.
